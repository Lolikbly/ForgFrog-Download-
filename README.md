# Huffman Secure Compressor & Image Encryptor

A comprehensive C++ security tool that provides data compression, message encryption, and secure image processing.

## Key Features
*   **User Authentication**: Robust Login and Registration system to protect user access.
*   **Dual Processing**:
    *   **Text Compression**: Uses the Huffman algorithm for lossless data reduction.
    *   **Image Encryption**: A dedicated module for secure image processing and protection.
*   **Strong Security**:
    *   **AES Encryption**: Messages and data are locked using the Advanced Encryption Standard.
    *   **SHA-256 Hashing**: Passwords for AES are securely hashed to prevent unauthorized access.

## Modules
*   **Compression Engine**: High-performance Huffman coding implementation.
*   **Encryption Suite**: AES-based encryption for both text and image assets.
*   **Identity Manager**: Secure storage and verification of user credentials.

## Tech Stack
*   **Language**: C++
*   **Algorithms**: AES, SHA-256, Huffman Coding
*   **GUI**: Windows Forms / Visual Studio

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com
   ```
2. Open `HuffmanApp.slnx` in **Visual Studio**.
3. Build and run the project (F5).
